# 🎪 開催概要

---

生成AIの活用は、「チャットで質問する」段階から、「自律的に判断し実行するAIエージェント」へと急速に進化しています。本ハッカソンは、皆さんに最前線の技術と開発プロセスを、実際に手を動かしながら体験できる機会を提供します。

**DevOps × AI Agent Hackathon** は、Google CloudのAI技術（Gemini Enterprise Agent Platform、Gemini等）を必須技術として、AIエージェントの企画から開発、デプロイ、運用までを一貫して実践できるハッカソンです。単なるプロトタイピングにとどまらず、実運用を見据えたDevOpsのプロセス全体をカバーします。

メインスポンサーにグーグル・クラウド・ジャパンを迎え、さらにツールスポンサー各社からの開発環境の提供や技術的なアドバイスを通じて、より実践的かつ高度なAIエージェント開発に取り組むことが可能です。

![Google_FROYO.png](attachment:53d3c0d9-9198-428b-9087-b9153a1b6d90:Google_FROYO.png)

8/19（水）に**Google 渋谷オフィス（渋谷ストリーム）**で開催される最終ピッチに審査で選ばれた10チームをご招待！そのアイデアを存分に発表いただきます。

**賞金総額は200万円**。AIを活用した実践的な開発に興味のあるソフトウェアエンジニアはもちろん、初めてハッカソンに参加される方や個人での参加も歓迎しています。ぜひあなたの技術とアイデアを形にし、次世代のAIエージェントの可能性に挑戦してください！

---

# 🚩 3つのコンセプト

- **つくる**
Google CloudのAIを中核に、実務で役立つ独創的なAIエージェントを設計・実装します。
- **まわす**
GitHub連携やCI/CDなどDevOpsのフローを構築し、AIを継続的に改善するサイクルを体験します。
- **とどける**
Cloud Runへのデプロイを通じ、スケーラブルな環境で本番品質のプロダクトをユーザーへ届けます。

# 🎓 学びをサポートする３つの仕掛け

- **🫱🏻‍🫲🏻チームビルディングイベントの開催**
    - チーム開発を経験したい個人参加者向けに、2026年6月7日（日）にチームビルディングイベントを開催します。個人・チームでのワークショップを通して、アイデアの発散からチームづくりまでをお手伝いします。
- **⛺ Boot Camp**
    - グーグル・クラウド・ジャパン主催のAI エージェント ハンズオンシリーズ「Agentic AI Bootcamp 2026」に加え、各ツールベンダーによる実践的なオンラインコンテンツを提供（無料受講／事前申込制）。参加者全員のベースラインを引き上げます。
- **💡 審査員によるフィードバック**
    - AIエージェント開発およびその周辺領域で実績を持つエンジニアが審査員として参加。提出作品に具体的なフィードバックを行います。
    - また、9月に開催予定のアフターイベント（オンライン）では、惜しくも入賞には至らなかったものの、優れた着想や技術が光る「審査員の推し作品」を取り上げ、審査員とともにその魅力や工夫を称えます。
