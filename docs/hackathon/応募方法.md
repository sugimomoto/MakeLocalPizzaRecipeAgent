# ⚙️ 応募方法

---

## STEP①：ハッカソンへの参加申し込み

[**Findy Conferenceの 申し込みフォーム**](https://conference.findy-code.io/preview/0bd90257-7e6a-4759-af21-8c63aaa0f52e/conferences/DevOps-AI-Agent-Hackathon/30/registration) からエントリー

個人で参加される方はもちろん、チームでの参加を希望される方も**全員のお申込みが必要**です。

## STEP②：作品をつくってProto Pediaに登録

本ハッカソンの審査には作品共有プラットフォーム「[Proto Pedia](https://protopedia.net/)」を利用します。

→　[Proto Pediaへの作品登録はこちら](https://protopedia.net/)

→　【ご参考】[**Proto Pediaの使い方**](https://protopedia.gitbook.io/helpcenter/registration)

### 【Proto Pediaへ作品を登録する際のルール】

| 入力項目 | 必須 / 任意 | 補足事項 |
| --- | --- | --- |
| 作品ステータス | 必須 | 任意のステータスを選択ください。登録後の変更も可能です。 |
| 作品タイトル | 必須 |  |
| 概要 | 必須 | 作品の概要をご記入ください。 |
| 画像 | 任意 | 作品の紹介画像がアップロードできます（最大5枚まで登録可） |
| 動画 | 必須 | YouTube もしくはVimeoにアップロードした動画のURLをご入力ください。 |
| システム構成 | 必須 | 「システムアーキテクチャ図」のアップロードを必須とし、技術的な補足があればご記入ください。 |
| 開発素材 | 必須 | 本作品の開発に使用した開発ツールをご記入ください。 |
| タグ | 必須 | 複数設定が可能です。1つ必須で「**findy_hackathon**」を設定してください。 |
| ストーリー | 必須 | ①本作品で解決したい課題とその背景　②想定する利用ユーザー　③プロダクトの特徴　をご記入ください。 |
| メンバー登録 | 任意 |  |
| 関連URL | 任意 |  |

## STEP③：作品提出フォームから最終応募

**作品提出フォーム（Google Form）**からの応募をもって、正式エントリー受付が完了します。

→　[作品提出フォーム（Google Form）](https://docs.google.com/forms/d/e/1FAIpQLScYR-nIwo2Fglx1Srlui2dDt5rN_iIS6YYeLfMrRHvUpoMuFg/viewform)

なお応募にあたって、**下記3点**の提出が必要となります。

1. GitHubリポジトリのURL（公開リポジトリ）
2. デプロイしたプロジェクトのURL（動作確認できる状態にしておくこと）
3. 「Proto Pedia」に登録した作品のURL

詳しくは、作品提出フォーム内の項目をご確認ください。
