# ユビキタス言語定義(Glossary)

> 本書は **ふるさとピザ帳** (技術名: MakeLocalPizzaRecipeAgent) で使用するドメイン用語・ビジネス用語・UI/UX 用語の正式な対応表を定義する。
> ドキュメント・コード・UI 文言・コミットメッセージ・コミュニケーションでこの用語に従う。
> 用語の追加・変更は基本設計の変更を伴うため、慎重に判断し本書を更新する。

---

## 1. 用語管理の方針

### 1.1 原則

1. **概念に対して呼称は一つ**(同じものを別名で呼ばない)
2. **ドメイン用語は日本語起点で考え、コード上は英語**(一貫した対応表を維持)
3. **UI に出る言葉は本書に掲載**(ユーザー向け文言を勝手に変えない)
4. **曖昧用語は使わない**("ユーザー" → "ホスト"、"フィードバック" は厳密に定義)

### 1.2 表記ルール

- 日本語: 漢字・ひらがな・カタカナの揺れを避ける
- 英語: 識別子は [development-guidelines.md §3](development-guidelines.md#3-命名規則) に従う(TS は camelCase、Python は snake_case)

## 8. 表記の揺れを避けるためのガイド

### 8.1 同義語の禁止と統一

| ❌ 使わない | ✓ 使う | 理由 |
| --- | --- | --- |
| ユーザー(曖昧) | ホスト | ゲストとの混同を避ける |
| AI / 生成 AI(曖昧) | エージェント / Gemini | 本プロダクトの中核は単なる AI ではなくエージェント |
| 評価(曖昧) | 総合評価 / 観点別評価 | フィードバック内の 2 種を区別 |
| テーマ(曖昧) | recipeTheme(料理の方向性。MVP では未使用) | UI テーマと混同しないため。本プロダクトに UI テーマ切替はない |
| 履歴 / 過去レシピ | ピザ帳(一覧画面) / 過去レシピ(個別) | 集合と個体を区別 |
| 写真 / 画像 | 写真(ユーザー撮影) / 画像(生成) | 出所が明確に違う |
| アプリ / サービス / プロダクト | プロダクト | 公式表現を「プロダクト」に統一 |
| 対話 / チャット / ウィザード | Quick Tap | 旧版概念を引きずらない。新仕様には対話・ウィザードは存在しない |
| 案 / 候補 | 候補(個別)/ 候補3案(集合) | レシピ確定前の提案を指す統一語 |
| 王道 / 安牌 / 鉄板 | Exploit (王道) | 戦略軸の日本語ラベルは「王道」一本 |
| 攻め / 冒険 / チャレンジ | Explore (大冒険) | 戦略軸の日本語ラベルは「大冒険」一本 |
| 微調整 / 改善 | Tune (一歩外す) | 戦略軸の日本語ラベルは「一歩外す」一本 |

### 8.2 数字・単位の表記

- 温度: `280°C`(度記号 + C)
- 時間: `8 分` / `30 秒`(数字 + 半角空白 + 単位)
- 数値: 三桁区切りなし(短いため)
- 通貨: `50万円` / `200 万円`(漢数字併用、ハッカソンページの表記に合わせる)

### 8.3 戦略軸の表記

コード上・ログ上・ダッシュボード上は **英語小文字 ID** を使う:

- ✓ `strategy: "exploit"` / `strategy: "tune"` / `strategy: "explore"`
- ❌ `戦略: "王道"` / `strategy: "Exploit"`

UI 上のユーザー向け表示は **日本語ラベル**:

- ✓ 「王道」「一歩外す」「大冒険」

## 9. 用語集の運用

### 9.1 追加・変更プロセス

1. 新しいドメイン概念を導入する PR の中で本書を同時更新
2. 既存用語の意味変更は基本設計レベルの変更とみなし、PRD・機能設計書も合わせて更新
3. 短期的に使う作業用語は本書には載せず、`.steering/` 内に留める

### 9.2 不整合発見時

- コード・ドキュメントで本書と異なる用語を発見したら、PR で修正する
- 用語の表記揺れは品質低下のサイン

### 9.3 旧プロダクト(MakePizzaRecipeAgent) からの廃止語

参考までに、旧版で使われていたが本仕様では廃止された用語:

| 旧語 | 廃止理由 |
| --- | --- |
| `dialogUx` / `chat` / `wizard` / `deck` | 対話型 3UX 切替を廃止し Quick Tap に一本化 |
| `agentSession`(対話セッション) | Quick Tap セッションに置換 |
| `turn`(対話ターン) | 多段対話を持たないため不要 |
| `theme`(UI テーマ craft/wa/editorial) | 和モダンの単一テーマに統一 |
| `recipeTheme` | Quick Tap では明示的入力を持たないため UI 上は未使用(将来 v2 で復活余地あり) |
